# Consumo de API - Super Heróis

## 📌 Descrição

Este projeto consiste em uma aplicação web desenvolvida para demonstrar o consumo de uma API utilizando JavaScript com programação assíncrona.

A aplicação realiza uma requisição HTTP para uma API pública de super-heróis, processa os dados retornados em formato JSON e exibe essas informações dinamicamente na interface da página.

Os dados são apresentados em formato de cards contendo informações dos personagens.

Este projeto foi desenvolvido como parte de uma atividade de sprint focada no consumo de APIs e desenvolvimento web utilizando HTML, CSS e JavaScript.

## 🌐 API utilizada

Inicialmente foi considerada a utilização da **Marvel API oficial**, porém o portal de desenvolvedores da Marvel foi descontinuado para novos acessos.

Por esse motivo foi utilizada uma API pública de super-heróis que reúne dados de diversos personagens, incluindo personagens da Marvel.

Endpoint utilizado:
https://akabab.github.io/superhero-api/api/all.json

A API retorna dados em formato JSON contendo informações como:

- nome do personagem
- imagem
- editora (Marvel / DC)
